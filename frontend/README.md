# Frontend README

This React app provides the user interface for the AI Tutor starter application.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

The app will be available at http://localhost:5173

## Features

- Login form
- Registration form
- Dashboard view after authentication
- API communication with the backend

## Environment

The frontend currently targets the backend at:

- http://localhost:5000/api
